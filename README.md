# Personal Portfolio Website - Yiğit Emir Cengiz

This project is an interactive, modern, and cyberpunk-themed professional portfolio website. It serves as a comprehensive showcase of my projects, technical skills, and experiences as a Computer Engineering student.

## Key Features

*   **Dynamic Internationalization (i18n):** The site is equipped with a custom-built system allowing users to switch instantly between English (ENG) and Turkish (TUR).
*   **Interactive Project Cards:** Each project features an animation mechanism that displays technical details and direct GitHub links upon interaction.
*   **Visual Atmosphere:** Created using pure HTML5 Canvas API, the site offers a nostalgic cyberpunk aesthetic featuring animated stars, a retro grid floor, and scanline effects.
*   **Fully Responsive:** The design is optimized for a seamless experience across all devices, including mobile phones, tablets, and high-resolution desktop monitors.

## Technical Stack

The following technologies were utilized in the development of this website:
*   **Frontend:** HTML5, CSS3
*   **Scripting:** Vanilla JavaScript (ES6+)
*   **Graphics:** HTML5 Canvas API (for background animations)
*   **Typography:** Google Fonts (VT323 & Segoe UI)

## Featured Project Categories

Through this portfolio, you can access my work in the following specialized fields:
*   **Machine Learning:** Predictive modeling using complex datasets, such as the Titanic Survival Prediction model.
*   **Data Visualization:** Converting analytical data into meaningful visual insights, demonstrated in the Netflix and Retail Sales analyses.
